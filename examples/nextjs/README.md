# Next.js + Brizy.io starter kit

## Getting Started

Pre-requisites

This guide will assume that you have the following software installed:

- nodejs (>=16.0.0)
- npm
- git

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# Nuxt.js + Brizy.io starter kit

## Getting Started

Pre-requisites

This guide will assume that you have the following software installed:

- nodejs (>=16.0.0)
- npm
- git

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

To learn more about Nuxt.js, take a look at the following resources:

- [Nuxt.js Documentation](https://nuxt.com/docs) - learn about Nuxt.js features and API.

You can check out [the Nuxt.js GitHub repository](https://github.com/nuxt/vercel-builder) - your feedback and contributions are welcome!
